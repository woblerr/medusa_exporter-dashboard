# Grafana dashboard for medusa_exporter

A dashboard of metrics produced by [medusa_exporter](https://github.com/woblerr/medusa_exporter).

To get this dashboard from **grafana.com**: [Medusa Exporter Dashboard](https://grafana.com/grafana/dashboards/24485-medusa-exporter-dashboard/).

To get this dashboard from **github.com**: [Medusa Exporter Dashboard](https://github.com/woblerr/medusa_exporter-dashboard).

## Compatibility with Medusa exporter versions

Depending on the version of the medusa_exporter, it is recommended to use different versions of the dashboard.

| Dashboard version | medusa_exporter version |
|---|---|
| `rev1` | `>= v0.2.0` |
